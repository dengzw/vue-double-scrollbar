# vue-double-scrollbar
>一个简单的vue横向双向滚动组件
## 安装

``` bash
npm install vue-double-scrollbar -s
```

## 使用

```js
import Vue from 'Vue'
import vueDoubleScrollbar from 'vue-double-scrollbar';
Vue.use(vueDoubleScrollbar)
```

## Example

```html

<vue-double-scrollbar>
  <div>
    ...your content
  <div>
</vue-double-scrollbar>

```
